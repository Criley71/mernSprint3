# PaxPal MERN app

PaxPal App following this tutorial (https://www.youtube.com/watch?v=-0exw-9YJBo) series on YouTube.

## Usage

Rename the .envexample to .env and add your MONGO_URI

### Install dependencies

```
# Backend deps
npm install

# Frontend deps
cd frontend
npm install
```

### Run Server

```
npm run server
```
